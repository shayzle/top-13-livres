# Top 13 Books I've Read · Shanisya Lahida

An interactive page showcasing the 13 books that have shaped the way I think.

Live: https://shayzle.github.io/top-13-livres/

---

## About

A single-page experience built with vanilla HTML, CSS, and React (via CDN).
Features a fan-style book cover carousel with smooth animations, swipe and scroll support.

---

## Features

- Drag, scroll, or use arrow keys to navigate
- Title, author and version (French or English) displayed for each book
- Fully responsive: works on mobile and desktop

---
